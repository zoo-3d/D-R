## 使用方法

鼠标用户：

`option/alt + 鼠标左键拖动`可以在窗口任意位置拖动窗口

`option/alt + 鼠标右键拖动`可以在窗口任意位置缩放窗口

触摸板用户：

`option/alt + 单指拖动`可以在窗口任意位置拖动窗口

`option/alt +  双指拖动`可以在窗口任意位置缩放窗口
